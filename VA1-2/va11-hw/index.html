HTML Head Tags

The head Element

the head element is a container for metadata(data about data ) and is placed between HTML tag and body tag

the title Element

the title element defines the title of the documents .it is shown in the bowser's title bar on the page tab. the page title is used to search emgine algorithms to decide the order when listning pages into search result .

The link Element

the link elemnt defines relationshop between curret documents and an exteral resource . the link tag is most often used to link external style sheet

The meta element

the meta element is typically used to spacify the character the charater set , page discription ,keywords,auther and documents and viewport setting the metaData will not be displayed on the page ,but is used by browser (how to display content) by search engines for the kaywords and other web sevics .

some of meta tags are here :

  • Define the character set used :

    <meta charset="UTF-8" />

  • Defines the viewport to make your website look good on all Devices :

    <meta name="viewport" content="width=device-width">

  • Defines the kaywords for the search engine :

    <meta name="Keywords" content="html,css,js">

  • Defines the auther of the web page :

    <meta name="auther" content="john doe">
